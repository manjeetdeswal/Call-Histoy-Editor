## Features

- Import and export call logs
- Import and export contacts
- Add custom call log entries
- Edit call history details:
    - Call date and time
    - Call type (Incoming, Outgoing, Missed, Rejected)
    - Call duration
    - Phone number
- Delete multiple call logs at once
- Backup and restore call history
- Search and filter call logs
- Simple and easy-to-use interface

## Use Cases

- Manage and organize call history
- Create sample call logs for testing
- Backup call logs before switching devices
- Restore call history from backups
- Test applications that rely on call log data
